##### 指定范围：

```
intitle: + 关键字	限定关键字在标题中
inurl: +关键字（.edu .gov等）
intext: + 关键字	文本
site: + 网站链接
```

##### 查找文件：

```
filetype: +关键字（pdf ppt等）
```

##### 时间限定：

```
搜索内容+空格+年份..年份
```
